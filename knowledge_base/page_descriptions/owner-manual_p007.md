# Page 7: Specifications Tables

This page contains three specification tables for the three welding process categories supported by the OmniPro 220. Each table has columns for the parameter name, 120VAC/60Hz values, and 240VAC/60Hz values.

## MIG Specifications Table

| Parameter | 120VAC/60Hz | 240VAC/60Hz |
|---|---|---|
| Power Input | 120VAC / 60Hz | 240VAC / 60Hz |
| Current Input at Output | 20.8A at 100A | 25.5A at 200A |
| Welding Current Range | 30-140A | 30-220A |
| Rated Duty Cycles | 40% @ 100A / 100% @ 75A | 25% @ 200A / 100% @ 115A |
| Maximum OCV | 86VDC | 86VDC |
| Weldable Materials | Mild Steel, Stainless Steel, Aluminum (with optional Spool Gun) | Same |
| Welding Wire Capacity | Solid Core: 0.025"/0.030"/0.035" -- Flux Cored: 0.030"/0.035"/0.045" | Same |
| Wire Speed | 50-500 IPM | Same |
| Wire Spool Capacity | Up to 12 lb spool | Same |

## TIG Specifications Table

| Parameter | 120VAC/60Hz | 240VAC/60Hz |
|---|---|---|
| Power Input | 120VAC / 60Hz | 240VAC / 60Hz |
| Current Input at Output | 20.6A at 125A | 15.6A at 175A |
| Welding Current Range | 10A-125A | 10A-175A |
| Rated Duty Cycles | 40% @ 125A / 100% @ 90A | 30% @ 175A / 100% @ 105A |
| Maximum OCV | 86VDC | 86VDC |
| Weldable Materials | Mild Steel, Stainless Steel, Chrome Moly | Same |

## Stick Specifications Table

| Parameter | 120VAC/60Hz | 240VAC/60Hz |
|---|---|---|
| Power Input | 120VAC / 60Hz | 240VAC / 60Hz |
| Current Input at Output | 19.5A at 80A | 23.7A at 175A |
| Welding Current Range | 10A-80A | 10A-175A |
| Rated Duty Cycles | 40% @ 80A / 100% @ 60A | 25% @ 175A / 100% @ 100A |
| Maximum OCV | 86VDC | 86VDC |
| Weldable Materials | Mild Steel, Stainless Steel | Same |

Note: The 240VAC input provides significantly higher maximum amperage across all processes (200A MIG, 175A TIG, 175A Stick) compared to 120VAC (100A MIG, 125A TIG, 80A Stick).
