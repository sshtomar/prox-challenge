# Page 12: Feed Roller Instructions

This page shows step 12 of MIG/Flux-Cored setup -- how to configure the Feed Roller for the correct wire type and diameter.

## Feed Roller Overview:
The Feed Roller is a dual-sided component. Each side has a groove machined for a specific wire type and diameter. The roller must be oriented so the correct groove faces the wire.

## Diagram Sequence (labeled A through D):

### Diagram A -- Feed Roller installed in mechanism:
Shows the Feed Roller Knob on top, securing the Feed Roller in the wire feed mechanism. Arrow indicates unscrewing the knob counterclockwise to remove it.

### Diagram B -- Solid Core V-Groove side:
Shows the feed roller removed, revealing two sides:
- One face: **0.030/0.035 groove** (V-shaped groove for solid core wire, 0.030" or 0.035" diameter)
- Other face: **0.025 groove** (V-shaped groove for 0.025" solid core wire)
The label "Feed Roller" points to the component itself. The text "Solid Core V-Groove" identifies this groove type.

### Diagram C -- Flux-Cored Knurled Groove side:
Shows the opposite roller orientation:
- One face: **0.045 groove** (knurled groove for 0.045" flux-cored wire)
- Other face: **0.030/0.035 groove** (knurled groove for 0.030" or 0.035" flux-cored wire)
The text "Flux-Cored Knurled Groove" identifies this groove type. Knurled grooves have a textured surface to grip the softer flux-cored wire without crushing it.

### Diagram D -- Reinstalling the roller:
Shows the Feed Roller being placed back into the mechanism and the Feed Roller Knob being screwed back clockwise to secure it.

## Critical Instructions:
- Check that the Feed Roller is correct for the type of wire being used (solid core or flux-cored)
- Confirm it is turned to properly match the wire size marked on the Wire Spool
- **The number showing on the roller must be the same as the wire diameter on the Spool**
- Flip or replace the Feed Roller as needed

## Wire Type to Groove Mapping:
- Solid core wire: Use the V-Groove side (smooth groove)
- Flux-cored wire: Use the Knurled Groove side (textured groove)
- Using the wrong groove type will cause feeding issues -- V-groove will slip on flux-cored wire, knurled groove will damage solid core wire
